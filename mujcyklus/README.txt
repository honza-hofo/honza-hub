MůjCyklus - Instalace na hosting
=================================

1. Nahrajte všechny soubory na hosting přes FTP
2. Vytvořte MySQL databázi (např. "mujcyklus")
3. Upravte config.php - zadejte DB přihlašovací údaje
4. Otevřete v prohlížeči: https://vasedomena.cz/install.php
5. Po úspěšné instalaci smažte install.php (bezpečnost!)
6. Hotovo - web je na https://vasedomena.cz/

Požadavky:
- PHP 7.4+ (doporučeno 8.0+)
- MySQL 5.7+ nebo MariaDB 10.3+
- mod_rewrite (Apache)
